C'est quoi le Phishing ?

Le Phishing est l’envoi de faux courriels non sollicités dans lesquels on demande aux destinataires, sous différents prétextes, de cliquer sur un lien menant vers un faux site Web ou d’ouvrir une pièce jointe malveillante. Le pirate y récupère les renseignements fournis et les utilise pour détourner des fonds à son avantage. Le faux site Web, tout comme le courriel, semble authentique, car il est souvent une copie conforme du site de l’institution financière ou de l’entreprise.

La raison principale pour laquelle ces attaques sont si réussies, c’est les utilisateurs eux-mêmes, puisque la plupart d’entre eux n’est pas suffisamment formé sur la façon de reconnaître le phishing. Ainsi, notre site web vise la sensibilisation aux menaces du phishing pour minimiser les pertes. nous vous proposons de prendre le test de niveau, pour tester vos connaissances et savoir jusqu'à quel point vous êtes capables à determiner si un mail est frauduleux. Nous vous proposerons aussi des devoirs à faire pour que vous puissiez améliorer vos connaissances sur ce sujet. Lire nos conseils Prendre le test de niveau
